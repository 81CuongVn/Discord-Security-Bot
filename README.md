# Discord-Security-Bot

An advanced Discord Security bot, including a ban database, watchlist, sticky messages, server lockdown, and embed builder. Supports multiple servers.

# Dependencies

```
- NodeJS V14+
- MySQL-Server
- Discord Developer Mode
- Discord Application
- Discord Server
```
